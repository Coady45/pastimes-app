# pastimes-app
Pastimes Web Application
 Project Overview

Pastimes is an online web application designed for buying and selling second-hand branded clothes that are still in good condition. The platform ensures quality by allowing administrators to review and approve items before they are visible to buyers. The system is designed to be user-friendly, secure, and enjoyable for both buyers and sellers.

 Features
 Buyer Features
Create an account and log in
Browse clothes with images, prices, and details
View product details
Message sellers directly
Add items to cart, edit, or remove them
Purchase items securely
 Seller Features
Register and log in
Upload clothing items (image, price, description, brand)
View and manage listed items
Communicate with buyers
Mark items as sold
 Admin Features
Approve or reject uploaded items
Remove sold or inappropriate items
Communicate with users
Manage overall system operations
 Innovative Features
Direct messaging system between buyers and sellers
Admin approval system to ensure quality control
Easy-to-use shopping cart
Automatic saving of delivery addresses
Clean and simple user interface
Technologies Used
Frontend: HTML, CSS
Backend: PHP
Database: MySQL
